完整流程速览

六步走一遍，每步用户在做什么、系统在做什么

1
灵感 · 看别家在投什么
输入品牌 / 域名 / FB 主页，把竞品广告抓回来，挑几条做参考收藏到参考板。
2
生成 · 一句话出图出片
写一段自然语言需求，Agent 多步规划，出 N 条候选；同一页面就能审核通过 / 驳回。
3
审核 · 把关
视频候选通过后默认进剪辑流；图片候选可直接存为资产，跳过剪辑。
4
剪辑 · 加 Logo 字幕 BGM 片尾
视频按 IR 渲染多规格；图片也可走剪辑做节日横幅式包装。一次设置，多规格批量出。
5
资产 · 入库带血缘
两种入库来源：剪辑出口 / 生成快捷路径。每条带完整版本血缘，可一键推到广告。
6
广告 · 跨平台投 + 数据回流
推到 FB / TikTok / Google，跨平台批量复制 campaign；每日拉数据回来标好坏素材。

6 个子页快速上手

灵感

看别家在投什么，收藏到参考板

  1. 顶部输入品牌名 / 域名 / FB 主页
  2. 过滤器选国家 / 投放期 / 素材类型
  3. 瀑布流点卡片看详情
  4. 心形一键收藏到参考板
  5. 板内多选 → "带去生成"
生成

一句话告诉 Agent，剩下交给它

  1. 大输入框写需求 / 卖点 / 平台
  2. 可附带参考素材或选模板
  3. 看 Agent 计划卡，必要时单步介入
  4. 候选通过 / 驳回 / 加备注
  5. 图片可选"存资产"快捷路径
剪辑

加 BGM / 字幕 / Logo / 片尾，多规格一次出

  1. 左侧 Tab 切换 BGM / 字幕 / Logo / 片头片尾
  2. 主舞台时间轴拖拽片段
  3. "保存为品牌包"下次一键复用
  4. 右侧勾选导出规格（1:1/9:16/16:9/4:5）
  5. 导出 → 进资产
资产

成品入库，看血缘、推到广告

  1. 左侧过滤树筛规格 / 状态 / 标签
  2. 网格 / 列表视图切换
  3. 点卡片看血缘、版本树、回流
  4. 多选 → 批量推到广告
  5. 详情可克隆为新版本
数据

看回流，挑好素材，淘汰差素材

  1. 顶部 KPI 卡看整体趋势
  2. 趋势图框选时间段细看
  3. 好素材榜"参照再生成"一键回灵感
  4. 差素材池"分析为什么不好"
  5. 调阈值定义好 / 差素材
广告

投放管理 + 跨平台复制 campaign

  1. 左侧账号树切平台 / 账号
  2. 三层 Tab：Campaign / Ad Set / Ad
  3. "+ 新建"走 7 步 Wizard
  4. 勾 Campaign → "复制到其他平台"
  5. 映射字段确认 → 创建草稿

从竞品到投放 · 完整流程

用别家爆款做参考，自己产一批多平台投放

  1. 到 灵感 搜竞品（如 "shein.com"），收藏 5-10 条到参考板
  2. 板内多选 → "带 N 条参考去生成"
  3. 生成页一句话需求，候选审核通过 → 进剪辑队列
  4. 剪辑页加品牌包默认字幕 / Logo / 片尾，导出 4 个规格
  5. 导出后自动进 资产 库，标签打好
  6. 资产页多选 → "推到广告"，Wizard 选目标平台 / 受众 / 预算
  7. 投放后第二天到 数据页看 CTR，差的下架，好的"参照再生成"

图片素材最快路径

图片走快捷路径，跳过剪辑直接入资产、直接投放

  1. 到 生成页大输入框写需求（如 "1:1 节日横幅，温馨氛围"）
  2. 会话流形态出 N 条候选，顶部确认 "图片默认走快捷路径" 已勾选
  3. 图片候选卡看到 存为资产 主按钮（右侧"推剪辑"是兜底）
  4. 点 "存为资产" → toast 提示 "已存为资产 · 去资产页 →"
  5. 资产页该条直接是 已发布 状态，可立即推到广告投放
  6. 若要给图片加 Logo / 包装成节日横幅，点右半"推剪辑"走老路径
为什么快：图片不需要 BGM、字幕、转场，剪辑层加工对它价值有限。直接入库省 2-3 步。

跨平台批量投放

一条素材同时投 Meta / TikTok / Google，少走重复路

  1. 到 资产页多选几条已发布素材
  2. 批量条 "推到广告" → 进 Wizard，第 2 步勾多个平台
  3. 或：已建好 Meta campaign 后到 广告页 Campaign Tab
  4. 勾选源 campaign → 批量条"复制到其他平台"
  5. 模态里选 TikTok / Google，字段映射检查表逐项确认
  6. ⚠ 行点 "去配置" 手动选投放目标 / 受众 / Pixel
  7. "创建草稿" → 目标平台账号下生成待发布草稿，再补完投放

常见问题

抓取的竞品素材能直接用于投放吗？

不能。竞品素材仅供参考——给 Agent 提供风格、构图、卖点的灵感。直接拿别家素材投放会有版权和广告平台合规问题。正确做法是收藏到参考板，带去生成页让 Agent 在你品牌包下产出"形似但完全原创"的版本。

生成的素材系统会自动判定合规吗？

系统会做规则前置过滤（违禁词、版权水印、人脸 / 名人脸检测），明显问题会拦掉、给候选卡打风险标签。但最终合规由人工审核环节把关。审核员通过的素材才进入剪辑或资产库。

跨平台复制后能修改吗？

能。跨平台复制会在目标平台账号下创建草稿状态的 campaign，可以在本平台或目标平台原生后台继续编辑、补完平台特有字段（受众、Pixel、出价方式等），确认无误后再正式投放。

资产的投放数据多久从平台拉回来？

默认日级（T+1），每天早 9 点统一更新前一天的数据。数据页顶部会显示 "数据 T+1" 提示。如果业务需要更高频，可以联系支持调整到小时级。

系统怎么自动判定"好素材"？

默认规则是 CTR ≥ 2.0% 且样本量 > 1000 次曝光。可以在数据页工具条齿轮里调整阈值（绝对值或同 campaign 内分位）。冷启动期样本不足时榜单会标 "样本不足，仅供参考"。

图片"快捷路径"和"推剪辑"该选哪个？

快捷路径（存为资产）：图片本身已可投放、不需要加 Logo / 字幕 / 包装时用，最便捷。
推剪辑：需要给图片加品牌 Logo、做节日横幅式包装、贴 CTA 二维码、多语言版本时用。两条路径都保留完整版本血缘。

预算到顶会发生什么？

生成页有会话级预算上限，到顶会暂停后续步骤，提示你"升一下预算"或"换便宜模型"。已生成的中间产物保留，不会丢失。

已下架的素材还能恢复吗？

资产软删除保留 30 天，期间可在 "已下线" 视图找回。30 天后物理删除（合规要求除外，长期保留的资产另作标注）。
